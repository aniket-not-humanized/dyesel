Important Notes Before Using Dyesel

Before You Start

To make the most out of Dyesel, here are a few key things you should know:

  1. System Requirements: Ensure your system can run Dyesel smoothly.
  2. Backup Important Files: While safe, always keep backups.
  3. Internet Connection: A stable connection is recommended for remote access.
  4. Security First: Configure your firewall to allow necessary ports.
  5. Admin Permissions: Running Dyesel may require administrator privileges.

Understanding Web Servers

A web server is a software that delivers web pages when requested by a user. Dyesel helps you set up a local web server quickly, making it easy to test and host websites.

What Are Ports?

Ports act as channels for network communication. Web servers commonly use port 80 (HTTP) or 443 (HTTPS).

Setting Up Port Forwarding

To allow external access, follow these steps:

  1. Log into your router's settings.
  2. Locate the Port Forwarding section.
  3. Set up forwarding for port 80 (or another as needed).
